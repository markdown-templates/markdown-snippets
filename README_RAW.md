## :zap: Usage

1. Go to the snippet that you want to use and click `Raw`  

![Raw button](https://i.imgur.com/QkHqujv.png)

1. Copy the code <kbd>Strg</kbd>+<kbd>C</kbd>
2. Paste the code in your markdown file <kbd>Strg</kbd>+<kbd>V</kbd>
3. _Done!_
